Sistema de Gerenciamento de Salas - Teleconsulta

Sistema de Gerenciamento de Salas de Espera para Teleconsulta

Cadastro, edição e exclusão de usuários do sistema

Cadastro completo de pacientes com dados pessoais e de saúde

Cadastro de unidades de saúde com CNPJ e CNES

Cadastro de salas por unidade e consulta de disponibilidade

Criação e cancelamento de reservas com validação de conflitos

Desenvolvido para gerenciamento de salas de espera em teleconsultas

Tecnologias: Java EE, JSF, PrimeFaces, Hibernate, Maven, Docker
